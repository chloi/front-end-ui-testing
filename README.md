
# front-end-ui-testing

> best practices for UI testing and repo structure


#### Features

- zero configuration
- headless UI testing (PhantomJS)
- simple application interation (NightmareJS)


#### Install Dependencies

```
brew install phantomjs
npm install
```

#### Usage

```
npm test
npm start
npm run depoy
```
